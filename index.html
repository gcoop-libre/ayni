Instalación

Puedes descargar y ejecutar ayni en gnu/linux usando los siguientes comandos:

sudo apt-get install git-core python-pygame
git clone https://github.com/gcoop-libre/ayni.git
python ayni/src/ayni.py

Acerca de ...

Ayni es una palabra Quechua, que se significa Ayuda mutua que se prestan las personas en el trabajo en Quechua
